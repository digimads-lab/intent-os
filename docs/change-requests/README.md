# Change Requests — 新增需求台账

本文件是所有新增需求（CR）的索引，每次新增需求时必须在此追加一条记录。

详细流程参见 `.claude/workflow.md` 的「新增需求流程（Change Request Flow）」章节。

## 状态说明

| 状态 | 含义 |
|------|------|
| 进行中 | CR 流程正在执行（CR-0 ~ CR-7） |
| 待回写 | CR-7 已通过验收，等待执行 CR-8 回写主文档 |
| 已完成 | CR-8 回写完成，人工确认无误 |
| 已搁置 | 需求暂停，不继续推进 |

---

## 如何新增一条需求

1. 在上方「CR 列表」追加一行，分配下一个编号（`CR-NNN`），状态填「进行中」
2. 创建目录 `docs/change-requests/CR-NNN/`
3. 在该目录下创建 `CR-NNN.md`，按模板填写需求描述
4. 按 workflow.md 中 CR-0 → CR-8 的步骤依次执行
